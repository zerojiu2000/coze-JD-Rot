# 求职雷达 · AI求职助手

专为转行者设计的 AI 求职工具，由 Coze 智能体驱动。

## 功能

- JD 匹配度评分
- 简历差距分析
- 打招呼话术生成
- 面试题预测

## 部署

1. Fork 或克隆本仓库
2. 导入 Vercel
3. 在 Settings → Environment Variables 添加：
   - `COZE_API_KEY` = 你的 Coze API Token
   - `COZE_BOT_ID` = 你的 Coze Bot ID
4. 保存并重新部署
